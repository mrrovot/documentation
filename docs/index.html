TurtlePay™ API Documentation

TurtlePay™ offers distinct Application Programming Interfaces for use by fellow developers depending on what you want to do and how you want to use that information.

The TurtlePay™ Payments Processing API

This API is designed for handling transactions in an easy to manage way. It is the showcase of the TurtlePay™ project.

For how you can interact with the Payments Processing API look no further than the API documentation.

The TurtlePay™ Blockchain Cache API

The majority of TurtlePay™ technology is powered by the Blockchain Cache API. It provides an open platform to fast, scalable, and reliable copies of the TurtleCoin™ blockchain without the need of operating a full node on the network.

For how you can interact with the Payments Processing API look no further than the API documentation.
